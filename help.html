About This Website

Accessibility Commitment

This website is designed to meet WCAG 2.1 AA accessibility standards. We've implemented screen reader compatibility, keyboard navigation, high contrast support, and clear content structure to ensure everyone can access our information.

Technical Information

Built with modern web technologies including HTML5, CSS3, and JavaScript. The site uses semantic markup, progressive enhancement, and follows web standards for optimal performance and accessibility.

Browser Support

Compatible with all modern browsers including Chrome, Firefox, Safari, and Edge. The website works on all devices and screen sizes with responsive design principles.

Mobile Experience

Fully optimized for mobile devices with touch-friendly navigation, readable fonts, and streamlined content. Dark mode support is available across all devices for better viewing comfort.
